Бинарные операции, конвертирование из/в 2,10,16чные системы счисления

Например имеем восьмибитное двоичное число:

01001011

каждый бит в этом бинарном числе умножим на соответствующий коэффициент и просуммировав получим следующие значения:

0* (2^7) + 1*(2^6) + 0*(2^5) + 0*(2^4) + 1*(2^3) + 0*(2^2) + 1*(2^1) + 1*(2^0) =

0 + 64 + 0 + 0 + 8 + 0 + 2 + 1 = 75

01001011(2) = 75(10)

для обратно преобразования число 75 будем делить на 2 и запоминать остаток отделения:

75 / 2 = (74 + 1) / 2 = 37 с остатком 1

37 / 2 = (36 + 1) / 2 = 18 с остатком 1

18 / 2 = (18 + 0) / 2 = 9 с остатком 0

9 / 2 = (8 + 1) / 2 = 4 с остатком 1

4 / 2 = (4 + 0) / 2 = 2 с остатком 0

2 / 2 = (2 + 0) / 2 = 1 с остатком 0

1 / 2 = (0 + 1) / 2 = 0 с остатком 1

теперь запишем число сверху вниз:

1001011(2) = (75)
