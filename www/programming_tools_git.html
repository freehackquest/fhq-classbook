Система контроля версий git

права на директорию ~/.ssh: 
	$ chmod 700 ~/.ssh
	$ chmod 655 -R ~/.ssh/*
	$ chmod 600 ~/.ssh/id_rsa
минимальный список файлов: 
	~/.ssh/config
	~/.ssh/known_hosts
	~/.ssh/id_rsa
	~/.ssh/id_rsa2
пример файла config: 
	Host git.yourserver.com
	 User git
	 IdentityFile ~/.ssh/id_rsa

	Host github.com
	 User git
	 IdentityFile ~/.ssh/id_rsa2
