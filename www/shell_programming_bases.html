Shell-программирование / Основы

Присваивание:

<переменная>=<значение>

Пример: 

	COUNT=30
	MSG="Hello, World!"

Условие:

	if <условие>; then
		<команды>
	elif/else <условие>; then
		<команды>
	fi

Цикл while:

	while <условие>
	do
		<команды>
	done

Цикл until:

	until <условие>
	do
		<команды>
	done

Условия/Числа:

  • Для n1 = n2 делаем так n1 -eq n2
  • Для n1 <> n2 делаем так n1 -ne n2
  • Для n1 > n2 делаем так n1 -gt n2
  • Для n1 >= n2 делаем так n1 -ge n2
  • Для n1 < n2 делаем так n1 -lt n2
  • Для n1 <= n2 делаем так n1 -le n2
Например нам нужно загрузить все номера журнала Хакера с сайта www.xakep.ru
Просто напишем скрипт!
	#!/bin/bash
	NUMBER=81
	COUNT=$1
	while [ $NUMBER -le $COUNT ]
		do
		if [ $NUMBER -le 9 ]; then
			ISSUE=00$NUMBER
		elif [ $NUMBER -le 99 ]; then
			ISSUE=0$NUMBER
		elif [ $NUMBER -gt 99 ]; then
			ISSUE=$NUMBER
		fi
		wget http://www.xard.ru/magazine/xh/$ISSUE/xh_$ISSUE.pdf
		let "NUMBER += 1"
	done

Пишем скрипт:
	#!/bin/bash
	dir="files/";
	echo "" > result.txt
	ls $dir | while read filename;
	#for file in $(ls "2_3_files/" | read )
	do
		cat "$dir""$filename" | grep -E " [0-9]{4,4}[a-zA-Z]{4,4} " >> result.txt

Осознать всю офигенность и мощь bash или shell программирования, но не забывайте нет универсальных языков, каждый предназначен для решения круга задач. Очень часто встречается комбинированные варианты.
